SimpleSlider

simple jQuery contents / images slider plugin

  • Download ZIP
  • Download TAR
  • View On GitHub

This project is maintained by amaroks

Features

  • Images Slides
  • Text / HTML Slides
  • Videos Slides
  • Slides Right/Left
  • Slides Up/Down
  • Control Speed
  • No CSS required
  • any slider height/width
  • Setup in 1 minute

Demo

  • Slide Right/Left
  • Slide Up/Down

a complete example

<!DOCTYPE html>
<html>
<head>
  <title>simpleSlider - Right/Left Demo</title>
  <script type='text/javascript' src='http://code.jquery.com/jquery-1.9.1.js'></script>
  <script type='text/javascript' src='js/jquery.simpleslider.js'></script>
</head>
<body>
  <div id="sliders" style="width:1024px; height:365px;">
      <div class="slide"> <img src="images/1.jpg"/> </div>
      <div class="slide"> <img src="images/2.jpg"/> </div>
      <div class="slide"> <img src="images/3.jpg"/> </div> 
  </div>

  <script type='text/javascript'>
    $("#sliders").simpleSlider();
  </script>
</body>
</html>

Change sliding direction from Left/Right to Up/Down

$("#sliders").simpleSlider({direction: "up" });

Change sliding direction and change sliding speed (in ms)

$("#sliders").simpleSlider({direction: "up", speed: 300 });
